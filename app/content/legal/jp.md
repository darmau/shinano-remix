# 利用規約

**最終更新日：2025年11月9日**  
本ウェブサイト（以下「本サイト」または「積薪（Firewood）」といいます）をご利用いただきありがとうございます。

本サイトにアクセス、登録、ログイン、またはサービスを利用した時点で、利用者は本規約のすべての条項に同意したものとみなされます。

## 第1条　本サイトについて

本サイトは個人により独立運営され、主にオリジナル記事の公開および閲覧・交流の機能を提供します。  
本サイトは広告配信やユーザーデータの販売などの商業活動を一切行いません。

---

## 第2条　データ収集およびプライバシー保護

1. **アクセス解析について**  
   本サイトは [Umami](https://umami.is) を利用し、サイトの改善を目的とした匿名のアクセス解析を行います。  
   収集される情報は以下の通りです。  
   - 訪問者の国または地域  
   - 使用デバイスの種類・OS情報  
   - ブラウザの言語・バージョン  
   - 訪問ページや滞在時間などの統計情報  

   これらの情報は**個人を特定しない形で集計**され、追跡や商業利用の目的で使用されることはありません。

2. **ユーザー認証について**  
   本サイトは [Supabase Auth](https://supabase.com/docs/guides/auth) を認証システムとして使用しています。  
   ログインや登録時の認可処理は Supabase のサーバー上で直接行われ、本サイトが取得するのは必要最小限の情報（例：ユーザーID、メールアドレス等）のみです。  
   本サイトはパスワードを保存しません。ログイン方法は以下の通りです。  
   - GitHub アカウントによるOAuthログイン  
   - メールで送信される Magic Link（ワンクリック認証）  

   上記の認証行為は Supabase および GitHub 各社のプライバシーポリシーに従います。

3. **Cookieおよびローカルストレージ**  
   ログイン状態の維持や利便性向上のため、本サイトでは Cookie またはローカルストレージを使用する場合があります。  
   これらのデータは本サイト内のみで使用され、第三者や他サイトへの追跡目的では利用されません。

## 第3条　アカウントとアクセス権限

1. ログインしたユーザーは、会員専用記事を閲覧できます。  
2. 未ログインユーザーは、公開記事のみ閲覧できます。  
3. ユーザーは自身のアカウントおよび登録メールアドレスの管理責任を負い、第三者への譲渡・共有を禁止します。

---

## 第4条　コメントおよびユーザー投稿

1. ログインユーザーのコメントは即時に表示されます。  
2. 匿名ユーザーのコメントは管理者の承認後に公開されます。  
3. コメント欄で「コメント返信通知を受け取る」にチェックを入れた場合、以下に同意したものとみなされます。  
   - コメントへの返信があった際、本サイトが指定メールアドレスに通知メールを送信すること。  
   - 上記の目的のために、メールアドレス情報を一時的に保存すること。  

4. ユーザーはコミュニティ基準に従い、以下の内容を含むコメントを投稿してはなりません。  
   - 暴力の扇動、ヘイトスピーチ、特定集団への差別  
   - 誹謗中傷、ハラスメント、プライバシー侵害  
   - 知的財産権の侵害  
   - 広告、スパム、無関係なリンクや投稿  

   違反コメントは削除され、悪質な場合はアカウントを永久停止することがあります。
